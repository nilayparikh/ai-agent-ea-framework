## Research Foundation

The LocalM-AiD framework is based on research synthesis from:

| Source                                           | Contribution                                 |
| ------------------------------------------------ | -------------------------------------------- |
| AI4SE Taxonomy (Schieferdecker, 2025)            | Classification of AI in software engineering |
| V-Bounce Model (Hymel, 2024)                     | AI-native SDLC methodology                   |
| Single Conversation Methodology (Escobedo, 2025) | Session management for AI interactions       |
| Ten Simple Rules (Poldrack et al., 2025)         | Practical coding guidelines                  |
| Metacognitive Framework (Phung et al., 2025)     | AI education principles                      |
